# Aplicação Web com JSF e Eclipselink

## Tecnologias

* Java 8;
* Arquitetura MVC;
* Design Pattern DAO;
* JavaServer Faces 2.2;
* Primefaces 6;
* JPA2(Eclipselink);
* Maven;
* REST(Gson e Jersey);
* Enterprise Javabeans 3;
* CDI;
* Servidor de Aplicação Glassfish 4.1;
* Banco de dados Oracle 11g;
* IDE Eclipse Oxygen.
